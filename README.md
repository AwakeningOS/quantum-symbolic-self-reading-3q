# quantum-symbolic-self-reading-3q

人生・思想・創作・霊的体験などを、3量子ビット・8状態の象徴回路として測定するブラウザツールです。入力と計算はブラウザ内で完結し、API、ビルド、外部依存はありません。

## 使い方

```bash
python3 -m http.server 8000
```

`http://localhost:8000` を開き、general または seeker の3Qエンコーダプロンプトで config JSONを作成し、サイトへ貼り付けて測定します。研究用の再現性確認には次を使えます。

```bash
node tools/encoder_variance.mjs config1.json config2.json config3.json
```

同じ物語を3回以上エンコードして分散を確認することを推奨します。

## 3量子ビットモデル

状態ベクトルは `a0,a1,b0,b1,c0,c1,d0,d1` の8成分です。ビット対応は `index = 4*q1 + 2*q2 + q3` で、時間軸は最下位ビットです。

- 主体軸 `q1`: general は当事↔世界、seeker は個我↔超越
- 顕現軸 `q2`: general は潜在↔顕在、seeker は非顕現↔顕現
- 時間軸 `q3`: 過去↔未来

| d0 | 帰結 | 過去から届いた出来事・結果・報い |

## ゲート族

- 表出系 `a*↔b*`、発現系 `c*↔d*`
- 共鳴系 `a*↔c*`、遭遇系 `b*↔d*`
- 転回系 `*0↔*1`
- 同時刻の役割系 `b*↔c*`、呼応系 `a*↔d*`

時間を跨ぐ対角も使用できますが、一つの出来事が複数軸を同時に跨ぐ理由を `meaning` に記述します。

## 三体の絡み合い

サイトは one-tangle、対タングル、Coffman–Kundu–Wootters の three-tangle を純粋状態から計算します。

| ラベル | 意味 |
|---|---|
| GHZ_KNOT | 三つの問いが一つの結び目として不可分 |
| W_WEAVE | 対ごとの粘り強い綾。二つずつほぐせる |
| HYBRID | 結び目と綾の混合 |
| SEPARABLE_LIKE | 三つの問いはほぼ独立 |

Bloch-z は全軸で正が前者（当事/潜在/過去）、負が後者（世界/顕在/未来）です。

## 時間を畳んだ視点

`projected_2bit` は時間ビットを周辺化し、`a=a0+a1` のように4成分へ投影した補助表示です。旧2ビット版とは別の3Q計算結果の要約であり、3Q状態を置き換えるものではありません。旧版は [quantum-symbolic-self-reading](https://github.com/AwakeningOS/quantum-symbolic-self-reading) を参照してください。

## 共鳴診断

`resonance_ratio = 反実仮想の最終L1重み / そのステップの即時L1変化` です。基準値1は後続干渉による増幅も相殺もない状態です。

| ラベル | 意味 |
|---|---|
| QUIET_SEED | 即時効果は小さいが後続との干渉で結末を大きく変えた |
| AMPLIFIED | 即時効果も大きく、最終重みはさらに増幅した |
| WASHED_OUT | 当時は大きかったが後の流れに洗い流された |
| PROPORTIONATE | 即時効果と最終重みが釣り合う |
| MINOR | 即時効果・最終重みとも小さい |
| DORMANT_BUT_STRUCTURAL | 即時効果はほぼゼロだが除去すると結末が変わる |
| NEGLIGIBLE | 即時効果・最終重みとも無視できる |

## 古典対照

- `phase_dependence`: 全ゲートの位相を0にした分布とのL1距離
- `interference_gap`: 干渉なし古典マルコフ遷移とのL1距離

両方LOWなら、そのconfigに複素振幅を使う経験的正当性は弱いと扱います。

## 回路だけが見えるもの

解釈AIには、反実仮想の重み、遅効性と相殺、順序依存性、通り方の質、三軸の絡み合い、時間の向き、予想と実測のずれを翻訳させます。サイト計算値の再計算・再分類は禁止しています。
